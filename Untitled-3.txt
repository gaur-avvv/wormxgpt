# 🚀 Getting Started

Welcome to the Open WebUI Documentation Hub! Below is a list of essential guides and resources to help you get started, manage, and develop with Open WebUI.

## ⏱️ Quick Start[​](https://docs.openwebui.com/getting-started/#%EF%B8%8F-quick-start)
[​](https://docs.openwebui.com/getting-started/#%EF%B8%8F-quick-start)
Get up and running quickly with our [Quick Start Guide](https://docs.openwebui.com/getting-started/quick-start).

## 🛠️ Advanced Topics[​](https://docs.openwebui.com/getting-started/#%EF%B8%8F-advanced-topics)
[​](https://docs.openwebui.com/getting-started/#%EF%B8%8F-advanced-topics)
Take a deeper dive into configurations and development tips in our [Advanced Topics Guide](https://docs.openwebui.com/getting-started/advanced-topics).

## 🔄 Updating Open WebUI[​](https://docs.openwebui.com/getting-started/#-updating-open-webui)
[​](https://docs.openwebui.com/getting-started/#-updating-open-webui)
Stay current with the latest features and security patches with our [Updating Open WebUI](https://docs.openwebui.com/getting-started/updating) guide.

## 🧪 Help Test Open WebUI[​](https://docs.openwebui.com/getting-started/#-help-test-open-webui)
[​](https://docs.openwebui.com/getting-started/#-help-test-open-webui)
Want to help improve Open WebUI? Run the [development branch](https://docs.openwebui.com/#using-the-dev-branch-)! Testing dev builds is one of the most valuable contributions you can make—no code required. Just use it and report issues you find.
Happy exploring! 🎉 If you have questions, join our [community](https://discord.gg/5rJgQTnV4s) or raise an issue on [GitHub](https://github.com/open-webui/open-webui).
